# Projeto DEVmoda (HTML, CSS, JavaScript e Gemini API)

## Sobre

Este projeto foi desenvolvido como parte da construção do meu portfólio durante a graduação em Sistemas de Informação.

O objetivo principal é servir como laboratório para estudar desenvolvimento Front-end moderno, responsividade, organização de código, consumo de APIs, integração com Inteligência Artificial e boas práticas de desenvolvimento web.

Além da evolução visual do projeto, todas as melhorias realizadas são documentadas através do histórico de commits no GitHub, permitindo acompanhar sua evolução desde uma landing page estática até uma aplicação web mais organizada e escalável.

## Melhorias Realizadas

### Interface

- Layout totalmente responsivo.
- Menu adaptado para dispositivos móveis.
- Banner com slider automático.
- Navegação por âncoras.
- Organização em seções independentes.
- Cards para apresentação das categorias.
- Integração com Google Maps.
- Links diretos para redes sociais.

### JavaScript

- Slider totalmente funcional.
- Menu mobile animado.
- Organização do código em funções independentes.
- Inicialização automática dos componentes.
- Redução de código duplicado.
- Estrutura preparada para reutilização em outras páginas.

### Inteligência Artificial

- Integração com a API Google Gemini.
- Geração de sugestões personalizadas de looks.
- Área de carregamento durante a requisição.
- Tratamento básico de erros da API.
- Sistema de repetição automática (Retry) para falhas temporárias.

### Organização

- Separação entre HTML, CSS e JavaScript.
- Padronização da estrutura dos arquivos.
- Organização dos recursos em diretórios.
- Código refatorado visando maior reutilização e manutenção.

## Como Executar o Projeto

Como o projeto é composto apenas por arquivos estáticos, existem diferentes formas de executá-lo.

### 1. Live Server (Recomendado)

Abra o projeto no Visual Studio Code e utilize a extensão **Live Server**.

### 2. Abrindo diretamente

Também é possível abrir o arquivo:

```text
index.html
```

diretamente no navegador.

### 3. Ambiente de Produção (GitHub Pages)

O projeto está publicado e disponível para demonstração através do **GitHub Pages**, permitindo visualizar sua interface e funcionalidades diretamente no navegador.

Link do projeto: [israel-reis-pereira.github.io/loja-megamoda/](https://israel-reis-pereira.github.io/loja-megamoda/)
